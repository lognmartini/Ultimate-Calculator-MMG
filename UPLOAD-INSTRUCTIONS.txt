HOW TO UPLOAD TO GITHUB (easiest)
  1. Go to https://github.com/lognmartini/mmg-calculator
  2. Click "Add file" → "Upload files"
  3. Open this folder in Finder: Desktop → Logan5-GitHub-Upload
  4. Press Cmd+A to select ALL files and the assets folder
  5. Drag everything into the GitHub upload box
  6. Commit message: "Logan5 wizard — desktop + mobile step flow"
  7. Click "Commit changes"

AFTER UPLOAD (Render will auto-deploy if connected)
  Live link will be something like:
  https://mmg-calculator-xxxx.onrender.com/go5.html
